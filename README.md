
# 太鼓六维Rating计算器

本工具是一个面向太鼓之达人玩家的“六维Rating”计算器，用于根据谱面及玩家表现估算六维 Rating 值，方便玩家分析成绩。

## 使用教程

- [B 站教程 by 卡璞·咕咕](https://www.bilibili.com/video/BV1u122BwEHQ)

## 快速开始

- [太鼓之达人 Rating & 六维分析](https://best.taiko.vanillaaaa.org)

## 主要功能

- **六维Rating 计算**: 根据谱面数据与玩家输入自动计算六项维度的 Rating 值。
- **自动填分**: 支持将数据库或爬取的分数导入并自动填充以批量计算。

## 文件说明

- **`src/`**: 源代码（Vue + TypeScript），包含视图组件与计算逻辑。
- **`public/songs.json`**: 歌曲数据库文件。
